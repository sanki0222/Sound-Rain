# Sound-Rain
The project was planned for Pin-up workshop...

Due to unexpected family circumstances, I had to go back to my home country for a week and was therefore unable to attend the Workshop 1 pin-up session in the church.

This repository documents the project that I originally intended to present during that workshop. The work itself was developed during the previous semester as a final project and was adapted for the context of the pop-up exhibition.

The repository includes the project code, relevant assets, and documentation demonstrating how the prototype runs and how the work was intended to be displayed.
